地球online 读博版 PWA 安装说明

把这个压缩包里的 5 个文件全部上传到 GitHub 仓库根目录：
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- apple-touch-icon.png

然后打开你的 GitHub Pages 地址，例如：
https://haji12138mi.github.io/p3/

手机端：
- Android Edge/Chrome：打开网页 → 菜单 → 添加到主屏幕 / 安装应用
- iPhone Safari：分享 → 添加到主屏幕

注意：
1. 第一次部署或修改后，GitHub Pages 可能要等 1–5 分钟。
2. PWA 有缓存。如果你更新了文件但手机 App 还是旧版，可以关闭再打开，或删除主屏幕图标后重新添加。
3. 已加入：每周日首次打开时进取心 -0.1，同一个周日刷新不会重复扣。
